# 时空探险 - 中国历史学习应用

> 版本 1.0 | 人教版七年级历史上册

## 项目简介

「时空探险」是一款专为中学生设计的历史学习应用，通过闯关模式、知识卡片、历史时间轴等创新功能，让历史学习变得生动有趣。

## 功能特性

### 🎮 闯关模式
- **20章 × 3关 = 60关**：每章分为初探、深入、终极挑战三个关卡
- **解锁机制**：通关自动解锁下一关，循序渐进
- **星级评定**：根据正确率评定0-3星
- **限时挑战**：已通关关卡可开启60秒限时模式

### 📚 题库系统
- **566道题目**：覆盖七年级上册全部20课
- **题目类型**：选择题 + 判断题
- **题目分类**：年份、人物、事件、制度、地点、意义等
- **题库隔离**：未解锁章节不出现在其他模式

### 📅 历史时间轴
- **73个历史节点**：从元谋人到北魏孝文帝改革
- **完整事件信息**：时间、地点、人物、起因、经过、结果、背景、目的、影响
- **关卡绑定解锁**：通关对应关卡解锁历史事件
- **交互式详情页**：点击节点查看完整历史信息

### 📜 知识图鉴
- **20张知识卡片**：每课一张，通关解锁
- **核心知识点**：5条要点 + 知识总结
- **收集进度**：展示解锁数量和进度

### 🔥 每日挑战
- **打卡系统**：每日答题打卡
- **连续天数**：记录连续打卡天数
- **奖励机制**：连续7天额外奖励

### 🤖 智能推荐
- **薄弱章节分析**：自动识别正确率低的章节
- **自适应难度**：根据答题表现调整题目难度
- **个性化推荐**：推荐需要加强练习的内容

### 📊 学习报告
- **正确率趋势**：可视化展示答题正确率变化
- **闯关进度**：已通关数、总星星、三星关卡数
- **知识点掌握度**：各章节正确率统计

### 🏆 成就系统
- **17个成就**：闯关、打卡、答题、连击、时间轴等类别
- **稀有度分级**：普通、稀有、史诗、传说
- **解锁提示**：达成条件自动解锁

### ✨ 视觉动效
- **答题反馈**：正确/错误动画效果
- **通关庆祝**：彩带飘落、星星爆发
- **解锁动画**：关卡解锁闪光效果

### 🔊 音效系统
- **答题音效**：正确/错误提示音
- **连击音效**：连续答对特殊音效
- **胜利音效**：通关/三星音效
- **背景音乐**：Web Audio API生成环境音

## 技术栈

- **前端框架**：Vue 3 + Vite
- **UI组件**：Vant 4
- **状态管理**：Pinia（持久化存储）
- **路由**：Vue Router
- **PWA**：支持添加到主屏幕
